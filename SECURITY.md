## Reporting a Vulnerability

Report security bugs by opening a new [Security Issue](https://github.com/HospitalRun/components/issues/new?template=security.md). You can also report a vulnerability by emailing security@hospitalrun.io.

Report security bugs in third-party modules to the maintainer or team maintaining the module.
